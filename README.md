# Customer Churn Analysis

This project analyzes telecom customer data to identify factors that drive customer churn and predict which customers are likely to leave.

Using Python, data analysis and machine learning techniques were applied to explore customer behavior and build a predictive churn model.

Key outcomes:
- Analyzed 7K+ customer records
- Identified major churn drivers (contract type, tenure, payment method)
- Built a predictive model with ~80% accuracy
- Generated insights to help businesses improve customer retention

Tools Used: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
